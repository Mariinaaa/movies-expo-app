# Movies Expo App – Guía de Ejecución
Explorador de películas con React y expo.
Esta es una guía rápida para ejecutar el proyecto en tu entorno local utilizando **React Native** y **Expo**.

## 1. Instalar dependencias

Ejecuta el siguiente comando dentro de la carpeta del proyecto:

npm install

## 2. Iniciar el servidor de desarrollo

npx expo start

 
Esto abrirá la interfaz de Expo en tu terminal o en tu navegador.

## 3. Ejecutar la aplicación
### Opción A: En tu teléfono
1. Instala Expo Go desde Play Store o App Store.
2. Escanea el código QR que aparece en la terminal o navegador.

### Opción B: En un emulador
1. Android: Abre Android Studio → AVD → Inicia un emulador
2. iOS (solo Mac): Presiona i en la terminal para abrir en el simulador de iOS
